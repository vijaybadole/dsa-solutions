# DSA Solutions — JavaScript

LeetCode and DSA solutions written in JavaScript with a focus on
multiple approaches, complexity analysis, and frontend-relevant patterns.

## Philosophy

Each solution includes:

- Brute force approach first
- Optimized approach with explanation
- Time and space complexity for each
- When to use this pattern in real frontend code

## Problem Categories

### Arrays & Strings

- Two pointer patterns
- Sliding window
- Prefix sums

### Trees & Graphs

- BFS/DFS traversals
- React component tree problems

### Dynamic Programming

- Memoization patterns (directly applicable to useMemo)
- Bottom-up tabulation

### Frontend-Specific

- Debounce and throttle implementation
- Deep clone and deep equality
- Event delegation patterns
- Promise chaining and async/await patterns

## Difficulty

Focused on medium-level problems — the level most commonly asked
at Tier-1 and Tier-2 product companies.


## Language

All solutions in JavaScript (ES6+). Chosen deliberately — the same
language used in frontend interviews and day-to-day work.
